# Procedimiento de Apertura de Expediente (EJEMPLO FICTICIO)

## 1. Identificación y conflicto de intereses
- [ ] Identificar al cliente (y, si aplica PBC, verificar identidad y titularidad real).
- [ ] Comprobar que no hay **conflicto de intereses** con otros clientes.

## 2. Encargo y honorarios
- [ ] Definir el alcance del encargo por escrito (hoja de encargo).
- [ ] Comunicar honorarios y suplidos previstos.

## 3. Apertura
- [ ] Crear el expediente en el sistema con un número único.
- [ ] Registrar plazos iniciales conocidos (ver control de plazos).
- [ ] Solicitar al cliente la documentación necesaria.

## 4. Confidencialidad
Toda la información del expediente está sujeta a **secreto profesional**. No se comparte con terceros
ni se procesa en sistemas no seguros con datos identificables.

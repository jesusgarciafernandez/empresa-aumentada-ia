# Skill: Evaluación de Necesidades

**Objetivo:** Analizar el perfil de un destinatario y detectar sus necesidades de forma preventiva.

## Cuándo se activa
- Preparación de visitas o reuniones.
- Auditoría de un cliente en cartera.
- Entrada de un cliente o caso nuevo.

## Entrada
- Sector, actividad o situación del destinatario.
- Tamaño o contexto relevante.
- Lo que ya tiene contratado o resuelto (si consta).

## Salida
- Mapa de necesidades principales y secundarias.
- Productos/servicios recomendables relacionados con el catálogo.
- Nivel de prioridad (urgente / importante / deseable).
- Preguntas clave a realizar en la reunión.
- Oportunidades (venta cruzada / mejora del servicio).

## Límites
La evaluación es orientativa y preventiva, no una auditoría vinculante.
Usa la plantilla `plantillas/informe_diagnostico.md` para presentar el resultado.

## Ejemplo
*Usuario:* "Evalúa las necesidades de este cliente nuevo con este perfil."
*Skill:* Lista priorizada de necesidades y preguntas clave para la reunión.

# Skill: Comparación de Opciones

**Objetivo:** Comparar productos, servicios u opciones de forma estructurada y dar una recomendación prudente.

## Cuándo se activa
- Se duda entre dos alternativas para un cliente.
- Se analiza una oferta de la competencia frente a lo que ofrece el negocio.
- Se compara una renovación vs. la opción actual.

## Entrada
- Opción A y Opción B (referencias documentales o datos aportados).
- Criterios de comparación (precio, características concretas o "comparación general").

## Salida
- Tabla markdown comparativa con las características principales.
- Diferencias clave (ventajas / limitaciones).
- Impacto para el destinatario de esas diferencias.
- Comentario profesional y prudente.
- Recomendación final fundamentada.

## Límites
La comparación se ciñe a la documentación aportada. Si faltan datos de alguna opción, debe avisarlo.

## Ejemplo
*Usuario:* "Compara el plan A con el plan B para este cliente."
*Skill:* Tabla comparativa resaltando dónde gana cada uno y una recomendación según el perfil.
